# ربات کاور عکس تلگرام
## معرفی و نحوه استفاده
این ربات عکس را از شما دریافت میکند و روی آن یک کاور قرار میدهد و برای شما می فرستد.
برای کاور عکس مد نظر خود را جایگزین عکس cover.png کنید.
همچنین تمام عکس هایی که برایش ارسال شده را ذخیره کرده و کنار هم قرار میدهد و یک کاور دیگر در روی آن میگذارد که برای تغییر این کاور فایل mycover.png را تغییر دهید.
## مشکلات رفع نشده
در این ربات هنوز مسئله ی نامساوی بودن سایز عکس های ارسال حل نشده است.
